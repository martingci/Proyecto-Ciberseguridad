## Componentes de la solución:

### Miner:
- Debe clonar el repositorio.
- Ejecutar análisis de código con CodeQL.
- Generar un SBOM con Syft.
- Detectar vulnerabilidades.

### Analyzer:
- Utiliza los datos del miner y se implementa por notebooks.

### Visualizer: 
- Debe permitir la distribución de vulnerabilidades, comparación de repositorios, comunicar diferentes en severidad y tipos.

### Aspectos adicionales
- Debe ser replicable por medio de _Dev Containers_.
